Simple CNN
===

Using C++ to build a simple CNN Project that can recongnize digits.

Build
---
```
make cmake
make build
```

Run
---
```
make run
```

Test
---
```
make test
```

Clean
---
```
make clean
```

Testing data
---
Please put at the test/data
Execute the program at the root folder
Put the output at test/data/tmp

Image Format
---
```
N(int, 4bytes)
M(int, 4bytes)
binary(Pixel, 1bytes/each)
```
